## ⚙️ PageIndex API Reference

The PageIndex API consists of two main components:

* [**PageIndex Tree Generation**](api/tree): Upload a document to generate a PageIndex tree index.
* [**PageIndex Retrieval**:](api/retrieval) Ask a query to retrieve relevant content from a processed document.

This documentation provides endpoints, parameters, and sample responses for both parts of the API.
