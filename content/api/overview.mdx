# Overview

The PageIndex API consists of two main components:

* [**PageIndex Tree Generation**](api-reference/tree-generation): Upload a document to generate a PageIndex tree index.
* [**PageIndex Retrieval**:](api-reference/retrieval) Ask a query to retrieve relevant content from a processed document.

This documentation provides endpoints, parameters, and sample responses for both parts of the API.
